

1. 项目概述（Logic Puzzle）
	•	将实体“逻辑拼图”玩具做成 桌面应用。
	•	技术栈：
	•	Rust

目标：
	•	提供一个可视化 8×8 棋盘 + 11 个彩色矩形块的解谜工具。
	•	支持玩家手动摆放、自动求解、简单随机关卡。

⸻

2. 游戏规则概要
	•	棋盘：固定 8 × 8 网格。
	•	方块：固定 11 个矩形块（均可旋转 90°），尺寸为：
	•	1×1、1×2、1×3、1×4、1×5
	•	2×2、2×3、2×4、2×5
	•	3×3、3×4
	•	每个方块有一个颜色（黑/蓝/红/黄/灰），仅用于显示。
	•	每一局：
	•	棋盘上放入3个黑色block作为“起始块”（固定位置，不可移动）；
	•	玩家需要使用其他所有block，填满所有非障碍格；
	•	不能越界、不能重叠，不能留下多余空格。

⸻


3. 目标
  	•	求解器： 给出一个已经放好起始块的8x8棋盘，求解一个或者多个解（或者无解）； （DFS 回溯是不是最简单的？）
	•	生成器： 生成初始棋盘（随机生成一个完整解， 从终局衍生关卡）
    •	可以给玩家游玩并判定解决
