

1. 项目概述（Logic Puzzle）
	•	将实体“逻辑拼图”玩具做成 桌面应用。
	•	技术栈：
	•	Rust：核心算法（求解、检测、关卡生成）。
	•	Tauri：桌面壳，负责前端 <-> Rust 通信。
	•	前端：HTML + CSS + TypeScript（可用或不用框架，随意）。

目标：
	•	提供一个可视化 8×8 棋盘 + 11 个彩色矩形块的解谜工具。
	•	支持玩家手动摆放、自动求解、简单随机关卡。

⸻

2. 游戏规则概要
	•	棋盘：固定 8 × 8 网格。
	•	方块：固定 11 个矩形块（均可旋转 90°），尺寸为：
	•	1×1、1×2、1×3、1×4、1×5
	•	2×2、2×3、2×4、2×5
	•	3×3、3×4
	•	每个方块有一个颜色（黑/蓝/红/黄/灰），仅用于显示。
	•	每一局：
	•	棋盘上放入3个黑色block作为“起始块”（固定位置，不可移动）；
	•	玩家需要使用其他所有block，填满所有非障碍格；
	•	不能越界、不能重叠，不能留下多余空格。

⸻
